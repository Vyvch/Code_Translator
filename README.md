# Создание плагина для IntelliJ Idea.

## Переводчик кода (Пивцаев Владислав Иванович M3100)

### Описание

В ходе лабораторной работы разработан и реализован плагин для IntelliJ Idea.
Плагин позволяет быстро переводить текст, выделенный в редакторе кода, через **Google Translate**, без использования API.

**Функциональные возможности:**

* Перевод выделенного текста на русский язык.
* Открытие перевода в браузере Google Translate.
* Горячая клавиша для быстрого перевода: **Ctrl+\ + Ctrl+T**.
* Действие доступно через контекстное меню редактора: **ПКМ → Перевести Выделенный Текст**.

Плагин полезен при чтении комментариев, документации или кода на иностранных языках.

### Использование

1. **Выделение текста:**

   * Выделите текст в редакторе → нажмите **Ctrl+\ + Ctrl+T** → откроется Google Translate с этим текстом.

2. **Без выделения:**

   * Появится окно с информацией: `Выделите текст для перевода!`.

## Примечания

* Требуется интернет и браузер для работы Google Translate.
* Плагин работает через браузер, API не используется.
