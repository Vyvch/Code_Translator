# Создание плагина для IntelliJ IDEA Community Edition.

## Переводчик кода (Пивцаев Владислав Иванович M3100)

### Описание

В ходе лабораторной работы разработан и реализован плагин для IntelliJ IDEA Community Edition.
Плагин позволяет быстро переводить текст, выделенный в редакторе кода, через Google Translate, без использования API Google Translate.

Плагин полезен при чтении комментариев, документации или кода на иностранных языках.

### Как использовать плагин?

   1. Выделите текст в редакторе
   2. Нажмите сочетание горячих клавиш **Ctrl+\ + Ctrl+T** или нажмите на выделенный текст **Правой Кнопкой Мыши** и в контекстном меню выберите опцию **Перевести Выделенный Текст**
   3. В браузере откроется сайт Google Translate с этим текстом и его переводом на русский язык.

### Примечания

- Требуется интернет и браузер для работы Google Translate.
- Плагин работает через браузер, API Google Translate не используется.

### Используемые технологии
- Язык программирования **Kotlin**
- Cистема автоматической сборки **Gradle**
- Библиотека **IntelliJ Platform SDK**
- Java класс **URLEncoder** (для кодирования выделенного текста в UTF-8)
- **Java AWT Desktop API** (для открытия браузера)
