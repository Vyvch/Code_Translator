# Создание плагина для IntelliJ IDEA Community Edition.

## Переводчик кода (Пивцаев Владислав Иванович M3100)

### Описание

В ходе лабораторной работы разработан и реализован плагин для IntelliJ IDEA Community Edition.
Плагин позволяет быстро переводить текст, выделенный в редакторе кода, через Google Translate, без использования API Google Translate.

Плагин полезен при чтении комментариев, документации или кода на иностранных языках.

### Использование

1. **Выделение текста:**

   * Выделите текст в редакторе → нажмите **Ctrl+\ + Ctrl+T** или через контекстное меню **ПКМ + Перевести Выделенный Текст** → откроется Google Translate в браузере с этим текстом и его переводом на русский язык.

2. **Без выделения:**

   * Появится окно с информацией: `Выделите текст для перевода!`.

### Примечания

* Требуется интернет и браузер для работы Google Translate.
* Плагин работает через браузер, API Google Translate не используется.
